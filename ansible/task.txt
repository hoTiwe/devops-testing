Техническое задание для MobilePark


Ansible
Задача: необходимо подготовить 3 типовых сервера. При знании Swarm, добавить ansible код для введения серверов в кластер Swarm с labels “worker”. Требование к серверам:
1. Использование пароля для входа по ssh - запрещено
2. Заходить под пользователем root - запрещено
3. На серверах должны быть учетные записи администраторов + технические (для примера добавляем admin-1, admin-2, admin-3, а технические ansible, autodeploy)
4. Должен быть установлен docker фиксированной версии, какая есть в стандартном репозитории Ubuntu.
5. Необходимый софт mc ncdu cifs-utils nfs-common

Требование к технической реализации: 
1. Использование Ansible
2. Использование Git

Никаких ограничений по разработке Ansible нет, требуется сделать так, как считаете нужным.
